MACHINE SPECIFIKATIONS

Universal Laser Cutter

Filer format: .ai .dxf
Work area: 1200 x 600 mm
Materials: paper, cardboard, HDF/MDF, plywood, wood, fabric, leather acrylic, POM.
Software: Adobe Illustrator

Safety:
Never EVER, leave the laser while cutting!
Never cut plastic that contains PVC!

Flexicam Viper CNC

Filer format: .dxf .dwg .sldprt .sat/.sab .iges .stl, Rhino: Parasolid
Work area: 1000 x 650 x mm
Materials: Wood, MDF/HDF, Plywood, Sika, Foam
Software: Alphacam (get software form Makerlab)

Safety:
Wear Protection glasses and ear protection.
Do not leave the CNC while it is running!
Contact staff before using

Roland iModella IM-10 (Not installed)

Filer format: -
Work area: 86 x 55 x 26 mm
Materials: Soft material such as resins, modelling board, jelutong, PCB laminate.

Ultimaker 2, 3 & S5, 3D Printer.

Filer format: .stl .obj .3mf
Work area: 215 x 215 x 200 (Extended: 215 x 215 x 300 )
Materials: PLA, ABS, Nylon, CPE, PVA
Software: Cura

3D Systems ZPrinter 450

Filer format: .stl, .VRML
Build Volume: 203 x254 x 203 mm
Materials: Superfine polymer printing powder.

Mimaki Textile Jet Textile Printer

Filer format: .tif, 180 dpi, CMYK
Max print width: 1500 mm
Materials: Various Cotton* and Silk*

*Fabric have to be bought in KEA POP UP Store as they are pre-coated with chemical for the printer process to work.

Epson T-Shirt Printer

Filer format : .png, RGB or CMYK, 300 dpi
Max print area: 406 x508 mm
Materials: The printer supports cotton and cotton blend garment cloth (minimum 50% perferably 100% cotton)
Software: Epson Garment Creator v1.4.2 (under Utilitis)

Graphtec Cutting plotter CE6000-60

Filer format:.ai, dxf
Max cut width: 550 mm
Materials: Vinyl
Software: Adobe Illustrator

Roland BN-20 Inkjet Printer/Cutter

Filer format: .ai, eps, RGB or CMYK
Max print width: 515 mm
Materials: Vinyl, Heat transfer, 3M Wrap Film
